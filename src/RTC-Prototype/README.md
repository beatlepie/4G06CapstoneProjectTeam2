# RTC Unity Client Prototype

Barebones implementation of a chat system using ASP.NET Core SignalR

## Dependency Setup

1. Make sure NuGetForUnity is installed. You can acces the package manager by going to Window > Package Manager in the main Unity Editor view.
2. Go to NuGet > Manage NuGet Packages and search for **Microsoft.AspNetCore.SignalR.Client**.
3. Install the package, any dependencies will be automatically installed as well.
